# 读者指南 · 德语法律丛书书目

> 本文面向**所有访客**（无需口令、无需登录）。页面顶部的「🔒 解锁编辑」等按钮默认隐藏，只有站长解锁后才会出现，不影响阅读。

## 这个站点是做什么的

本站点收录两家著名德语法学学术出版社的系列丛书书目：

- **Mohr Siebeck**（莫尔·齐贝克）
- **Duncker & Humblot**（邓克与洪布洛特）

每一条目包含：**卷号 / 作者 / 德文原名 / 中文参考译名 / 出版年 / 标签**，并附**出版社商品页链接**，方便跳转查阅原书信息。

中文译名为**参考性译名**（由术语词典自动生成并优化），用于快速浏览与定位，正式引用请以德文原名为准。

## 如何开始浏览

1. 打开首页（`index.html`）。
2. 首页按出版社分区（Mohr Siebeck / Duncker & Humblot），列出各套丛书卡片，卡片上显示该丛书**册数**。
3. 点击任意丛书卡片，进入该丛书的详细书目页。

当前收录 8 套丛书，合计约 **2040 册**：

| 出版社 | 丛书 |
|---|---|
| Mohr Siebeck | Veröffentlichungen zum Verfahrensrecht（程序法出版物）、Jus Privatum（私法丛书）、Beiträge zum ausländischen und internationalen Privatrecht（外国与国际私法论丛）、Beiträge zum Arbeitsrecht（劳动法论丛） |
| Duncker & Humblot | Schriften zum Prozessrecht（诉讼法丛书）、Studien zum vergleichenden Privatrecht（比较私法研究）、Schriften zum bürgerlichen Recht（民法丛书）、Schriften zum Sozial- und Arbeitsrecht（社会法与劳动法丛书） |

## 丛书页表格怎么看

进入丛书页后，是一个按**卷号升序**排列的表格：

| 列 | 说明 |
|---|---|
| 卷 | 丛书卷号（蓝色加粗） |
| 作者 | 作者姓名 |
| 书名（德文） | 德文原名；**可点击**，在新窗口打开出版社商品页 |
| 中文译名 | 参考性中文译名（绿色），由术语词典生成 |
| 年 | 出版年（若有） |
| 标签 | 主题标签（见下方「标签」说明） |

- 顶部搜索框可实时过滤（见下）。
- 顶部显示「共 N 册 / 显示 M 册」的计数。

## 搜索

首页与各丛书页顶部均有搜索框：

- 输入关键词后，表格按 **书名 / 作者 / 标签** 实时过滤，无需回车。
- 右侧计数同步显示当前匹配册数。
- 清空搜索框即可恢复全部书目。

## 标签（Tag）

标签是对书目主题的人工/半自动标注，用于跨书、跨丛书检索：

- **首页标签云**：以字号大小体现标签出现频次，点击任一标签，下方会列出**所有含该标签的书**（按丛书顺序排序），实现跨丛书主题检索。
- **丛书页标签**：每本书右侧的标签 chip 也可点击，等价于在标签云中筛选。
- 标签是站长维护的辅助检索手段，可能不完整，仅供参考。

## 在手机上阅读

站点为响应式设计：

- 屏幕宽度 ≤ 760px：表格字号缩小、列宽自适应。
- 屏幕宽度 ≤ 480px：隐藏「作者」等次要列，聚焦书名与译名。

建议横屏或适当放大以获得最佳阅读体验。

## 数据更新说明

- 书目信息会不定期进行更新（页面顶部标注最近更新时间）。
- 中文译名、出版年、标签若有修订，由站长发布后**所有人可见**；在发布前仅站长本机可见。
- 出版社商品页链接指向外部站点，若原书下架或改版，链接可能失效，请以出版社官网为准。

## 常见问题

**Q：为什么有些书没有中文译名 / 出版年？**
A：译名与年份由站长逐步补全，暂缺的条目会留空，不影响其他信息查看。

**Q：我能直接修改译名或加标签吗？**
A：普通读者不能。编辑功能（✎ / 🏷️ / 拖拽）默认隐藏，需站长解锁。如需补充或更正，可联系站长。

**Q：德文书名点的链接打不开？**
A：链接指向出版社官网商品页，可能受网络环境影响；也可能该书已下架。
