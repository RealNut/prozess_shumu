这个站点是做什么的

本站点收录两家著名德语法学学术出版社的系列丛书书目：Mohr Siebeck（莫尔·齐贝克）与 Duncker & Humblot（邓克与洪布洛特）。每一条目包含：卷号 / 作者 / 德文原名 / 中文参考译名 / 出版年 / 标签，并附出版社商品页链接。

中文译名为参考性译名（由术语词典自动生成并优化），用于快速浏览与定位，正式引用请以德文原名为准。

如何开始浏览

  1. 打开首页（index.html）。
  2. 首页按出版社分区（Mohr Siebeck / Duncker & Humblot），列出各套丛书卡片，卡片上显示该丛书册数。
  3. 点击任意丛书卡片，进入该丛书的详细书目页。

顶部显示「共 N 册 / 显示 M 册」的计数，并可在搜索框中实时过滤。

搜索

首页与各丛书页顶部均有搜索框：

  • 输入关键词后，表格按 书名 / 作者 / 标签 实时过滤，无需回车。
  • 右侧计数同步显示当前匹配册数。
  • 清空搜索框即可恢复全部书目。

标签（Tag）

标签是对书目主题的人工 / 半自动标注，用于跨书、跨丛书检索：

  • 首页标签云：以字号大小体现标签出现频次，点击任一标签，下方会列出所有含该标签的书（按丛书顺序排序），实现跨丛书主题检索。
  • 丛书页标签：每本书右侧的标签 chip 也可点击，等价于在标签云中筛选。

示例标签 标签是站长维护的辅助检索手段，可能不完整，仅供参考。

在手机上阅读

站点为响应式设计：

  • 屏幕宽度 ≤ 760px：表格字号缩小、列宽自适应。
  • 屏幕宽度 ≤ 480px：隐藏「作者」等次要列，聚焦书名与译名。

建议横屏或适当放大以获得最佳阅读体验。

常见问题

为什么有些书没有中文译名 / 出版年？

译名与年份由站长逐步补全，暂缺的条目会留空，不影响其他信息查看。

我能直接修改译名或加标签吗？

普通读者不能。编辑功能（✎ / 🏷️ / 拖拽）默认隐藏，需站长解锁。如需补充或更正，可联系站长。

德文书名点的链接打不开？

链接指向出版社官网商品页，可能受网络环境影响；也可能该书已下架。
